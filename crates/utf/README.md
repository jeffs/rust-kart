# utf

Prints Unicode characters from codepoints or named characters.

## Usage

```sh
utf 1f600
# Output: (grinning face emoji)

utf frog poop lol
# Output: (frog, poop, laughing emoji)

utf command shift
# Output: (command and shift symbols)
```

## Named Characters

- `cent`, `command`/`cmd`, `shift`
- `frog`, `facepalm`, `grimace`, `horns`, `lol`, `ok`, `sob`, `poop`
- `up`
